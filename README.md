# Delta Robot-Based 3D Printer

A fully functional Delta Robot-based FDM 3D printer designed and built from scratch as my Bachelor's Thesis in Robotics Engineering.

The project covers the complete engineering process, from mechanical design and component selection to electronics integration, firmware configuration, calibration and validation.

---

## Overview

This project was developed to demonstrate the complete design and implementation of a Delta Robot used as a high-speed 3D printer.

The printer was entirely designed, assembled and calibrated by me, including:

- Mechanical design
- CAD modelling
- Electronics integration
- Firmware configuration
- Motion calibration
- Functional testing
- 3D printing validation

---

## Main Features

- Delta robot architecture
- Fully custom mechanical design
- CATIA V5 CAD models
- Klipper Firmware
- Raspberry Pi 4
- BigTreeTech Octopus Pro
- TMC2209 Stepper Drivers
- Optical Endstops
- ADXL345 Input Shaping
- PLA & PETG Printing
